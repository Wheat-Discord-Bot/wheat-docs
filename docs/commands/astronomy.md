# Astronomy

In this documentation, we will use default prefix is `e`.

## APOD

> Show a astronomy picture of specific date
>
> #### Syntax
>
> `eapod [DD/MM/YYYY]`
>
> If you don't provide a date, default will be that date.
>
> The date must after 16/06/1995
>
> #### Aliases
>
> `none`
>
> #### Example
>
> `eapod 09/12/2020`
>
> `eapod`
